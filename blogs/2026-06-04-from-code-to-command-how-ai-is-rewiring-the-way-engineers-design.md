Bentley Systems is advancing infrastructure engineering through AI coding tools that automate workflows via Python scripting, and Model Context Protocol (MCP) servers enabling AI agents to control engineering software using plain English commands. A demonstrated use case showed a 40% weight reduction in steel structures, though engineers must remain responsible for validation to ensure safety and design integrity.
